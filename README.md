# Lina Dark

如果不关灯的话容易招来BUG~

LinaDark 是 JumpServer 前端 UI 项目 Lina 的 深色主题 Fork项目。

自称CE版黑化程度99%,欢迎issues

## 生产中部署
打开 [RELEASE](https://github.com/Boc-chi-no/lina-dark/releases/) 页面, 找到对应版本根据releases上的命令安装

卸载进入安装文件夹执行./setup.sh uninstall
